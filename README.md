# 방학 마스 코테 방  
**GitHub Desktop 다운로드 받기!**  
다운로드 이후에 해야할 것들  
1. 레포지토리 들어가서 Fork버튼 눌러서 C든 D든 드라이브에 추가하기.
2. 코테 문제를 푼 후 본인 아이디로 된 파일에 txt파일을 추가 후 txt파일 안에 푼 코테 문제를 복사해 넣는다.
3. txt파일을 github desktop에서 파일을 연 뒤 commit 명령어를 따르면서 push & pull을 하여 오가니제이션에 올린다.

---

commit 명령어
```
git commit -m "[문제 사이트 이름] 제목 / 난이도"
```
